# tocabi_hand_ui

## To Run
```console
$ rosrun tocabi_hand_ui tocabi_hand_ui.py 
```
or
```console
$ cd {your_workspace}/tocabi_hand_ui/scripts
$ python3 tocabi_hand_ui.py
```

## Control Tab
<img src="https://github.com/user-attachments/assets/04a3b40b-1a58-4793-8b28-92cb15305e77" width="60%" height="60%"/>

You have two control options (current / current position) for grasping at each joint (AA / FE)


## Setting Tab
<img src="https://github.com/user-attachments/assets/78945a6f-dee8-4a33-9efa-5db4a768df05" width="60%" height="60%"/>
